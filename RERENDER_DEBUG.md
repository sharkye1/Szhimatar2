# Отладка кнопок переповтора рендера

## Что было изменено

1. **Текст кнопок** временно изменен на `R1` и `R2` вместо символов ↻ и ↻2
   - Это для проверки, видны ли кнопки вообще
   - Если вы видите R1 и R2 - иконки работают, но может быть проблема с символами
   - Если вы не видите R1 и R2 - проблема в другом месте

2. **Стили кнопок** сделаны более видимыми:
   - Добавлены border: 2px solid currentColor
   - Opacity повышена с 0.6 до 1
   - Добавлены display: inline-flex для правильного выравнивания

3. **History-item CSS** изменены:
   - Изменен align-items с flex-start на center для правильного вертикального выравнивания
   - Добавлен gap: 8px для правильного расстояния между элементами

## Как проверить

1. Откройте приложение и перейдите в раздел Statistics
2. Убедитесь, что фильтр установлен на "all" (не на другие фильтры)
3. Откройте DevTools (F12 или Ctrl+Shift+I)
4. В консоли найдите сообщение: `[StatisticsPanel] Loaded. Total renders: X Filtered renders: Y Filter: all`
   - Если Total renders = 0, то история вообще пуста
   - Если Total renders > 0 но Filtered renders = 0, проверьте фильтр
5. Если история видна, но кнопки не видны, проверьте:
   - В CSS нет ли скрытого overflow
   - Нет ли белого текста на белом фоне (проблема с color)
   - В браузере проверьте элемент кнопки в DevTools (Inspect Element)

## Места, где могут быть проблемы

1. **StatisticsPanel.tsx** - строка ~321
   - Условие: `render.status === 'completed'`
   - Если статус другой - кнопки не появятся
   - Проверьте в DevTools какой статус у элементов в истории

2. **StatisticsPanel.css** - классы:
   - `.history-actions` - контейнер для кнопок
   - `.re-render-btn` - стили кнопок R1/R2
   - `.delete-btn` - стили кнопки удаления

3. **Локализация** - если вместо "R1" видна "R1" но no title появляется
   - Проверьте что ключи `history.re_render_overwrite` и `history.re_render_new` определены в языковых файлах

## Как вернуть иконки (после проверки)

Когда убедитесь, что кнопки видны, измените текст на символы:

```tsx
// Вместо R1 и R2:
↻      // для первой кнопки (переиспользование)
↻2     // для второй кнопки (новая версия)
```

## Консоль браузера

При клике на кнопку в консоли должны появиться:
- `[StatisticsPanel] Re-render queued (overwrite): /path/to/video.mp4`
- или `[StatisticsPanel] Re-render queued (new version): /path/to/video.mp4 → /path/to/video_2.mp4`

Если их нет - клик не срабатывает, проверьте что кнопка действительно кликаема.
